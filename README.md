# Hippy_Front

## About

2021 우리은행 온택트 해커톤 프론트 프로젝트입니다.

## Skills

- React
- Redux, Redux-saga
- material ui

### Material UI : 5.0.0

2021년 4월 28일 기준 material ui의 알파버전입니다. sx라는 프로퍼티가 추가되어 쉽게 커스텀 스타일링을 해줄 수 있게 되었습니다.

## How to start?

- json-server를 설치한다.
  - npm install -g json-server
- json-server ./data.json --port 4000
- yarn start
